- RBAC for sellers and buyers
- Sell Products

## Tech Stack

- Frontend:
  - React.js
  - Zustand
  - React Query
  - ShadCN
  - Tailwind
  - Multer
- Backend:
  - Node.js
  - Express
  - Database (PostGres)
  - Prisma (ORM)
  - JWT Authentication
  - RBAC (Role Based Access Control)
  - RESTful API architecture
  - Docker (for development)
- Devops:
  - Shell Scripting
  - SSH
  - SSL/TLS
  - TCP/IP
  - UFW
  - Nginx(Web server)
  - Reverse Proxy (using Nginx)
  - DNS
  - PM2
  - CI operations with Git Actions
